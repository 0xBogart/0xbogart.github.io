SCRIPT

GRABAR SESIONES DE TERMINAL EN TIEMPO REAL

script es una herramienta que realiza una composición tipográfica de todo lo que hay en tu sesión de terminal.

Los datos de la terminal se almacenan en bruto en el archivo de registro y la información sobre la temporización en otro archivo de registro estructurado {opcional}

El archivo de registro de tiempo es necesario para reproducir la sesión posteriormente mediante scriptreplay y para almacenar información adicional sobre la sesión.

El comando exit o presionar las teclas Ctrl+D finalizarán la grabación de sesión.
☣ INICIAR GRABACIÓN

☣  script --timing=time_log sesion-1
Script starded, output log file is 'sesion-1'

☣ SCRIPTREPLAY

Esta herramienta repite una secuencia de comandos, utilizando la información de tiempo para asegurar que la salida se produce al mismo tempo que originalmente se grabó la sesión.

⦾ --timing: Archivo de registro de temporización de secuencias de comandos
⦾ sesion-1: Nombre de la sesión previamente grabada

☣  scriptreplay --timing=time_log sesion-1
