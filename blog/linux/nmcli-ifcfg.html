Añadir múltiples IP a una Interfaz de Red

network-scripts & nmcli



☣ MÉTODO 1: network-scripts
·  ip a  La interfaz enp0s3 tiene una dirección IPv4 e IPv6 configuradas por DHCP. 
·  vim /etc/sysconfig/network-scripts/ifcfg-enp0s3:1  Crear y configurar un script con el mismo nombre que el de la interfaz original, pero acabado en  :1 
·  ifdown enp0s3 && ifup enp0s3  Reiniciar la interfaz para aplicar los cambios. 


☣  ip a
enp0s3: <BROADCAST,MULTICAST,UP,LOWER_UP> mtu 1500 
link/ether 08:00:27:d8:63:0e brd ff:ff:ff:ff:ff:ff 
inet 192.168.1.52/24 brd 192.168.1.255 
    valid_lft forever preferred_lft forever 
inet6 fe80::a00:27ff:fed8:630e/64 


☣  vim /etc/sysconfig/network-scripts/ifcfg-enp0s3:1

DEVICE=ifcfg-enp0s3:1
TYPE=Ethernet
ONBOOT=yes
BOOTPROTO=none
IPADDR=192.168.1.45
PREFIX=24


☣  ifdown enp0s3 && ifup enp0s3 
☣ MÉTODO 2: nmcli {IPv4}

nmcli es una herramienta en línea de comandos para el control de NetworkManager y reportes de estado de la Red

·  nmcli connection show   Mostrar las conexiones de red IPv4 
·  nmcli connection modify +ipv4.address 192.168.1.45/24   Añadir una dirección IPv4 secundaria a la interfaz enp0s3  
·  nmcli connection reload   Reiniciar servicio de Red 
·  nmcli connection show enp0s3   Verificar que la configuración se ha actualizado 
·  ifdown enp0s3 && ifup enp0s3   Reiniciar la interfaz para aplicar los cambios 


☣  nmcli connection show
NAME   UUID                             TYPE    DEVICE
enp0s3 0eff2e08-8562-4965-19b517237ba1 ethernet enp0s3 


☣  nmcli connection modify +ipv4.address 192.168.1.45/24 
☣  nmcli connection reload 


☣  nmcli connection show enp0s3

☣ MÉTODO 2: nmcli {IPv6}

·  nmcli connection modify enp0s3 ipv6.method manual ipv6.addresses fd01::100/64   Añadir dirección IPv6. 
·  nmcli connection reload  Actualizar la configuración de Red. 
·  nmcli connection show enp0s3  Comprobar que nmcli ha actualizado los cambios y reconoce la dirección IPv6 secundaria 
·  ifdown enp0s3 && ifup enp0s3  Reiniciar la interfaz para aplicar los cambios. 


☣  nmcli connection modify enp0s3 ipv6.method manual ipv6.addresses fd01::100/64 
☣  nmcli connection reload
